# NewsShelf Frontend (React + Bootstrap)

Це готовий фронтенд під ваш бекенд (UserService, Search/NewsService, AdminService, RecService).
Підтримує:
- реєстрацію / логін (JWT)
- стрічку новин + деталізацію + лайки + коментарі
- пошук/фільтри/сортування
- профіль
- адмін-панель (керування новинами/користувачами, модерація коментарів, health/status)

> Важливо: через те, що контракти API (ендпоїнти) можуть відрізнятись, **усі URL винесені в .env**. Якщо у ваших сервісів інші маршрути — відкоригуйте файл `src/config/appConfig.js` та відповідні файли в `src/services/*`.

## Швидкий старт

1) Відкрийте папку в VS Code  
2) Створіть `.env` (можна скопіювати з `.env.example`) і виставте URL ваших сервісів  
3) Встановіть залежності та запустіть:

```bash
npm install
npm run dev
```

## Мок-режим (без бекенду)

У `.env` виставте:

```
VITE_USE_MOCK=true
```

## Ролі
UI підтримує ролі: `User`, `Moderator`, `Admin`.  
Для доступу до адмін-роутів використовується `user.roles` з профілю (див. `AuthContext`).
